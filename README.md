# FactorialCalculator


This is a simple C# console application that calculates the factorial of six numbers entered by the user. It handles negative inputs by setting them to 0 and warns if numbers are too large.

## How to use

1. Clone or download the repository.
2. Open the project in Visual Studio or any C# IDE.
3. Run the program.
4. Enter six numbers when prompted.
5. See the factorial results printed in the console.
